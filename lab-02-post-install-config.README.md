# Configure Windows Update
Step 2: Configure Windows Update

Open Settings > Windows Update (left sidebar).
Click Advanced options.
Under Active hours, set the range to 8:00 AM – 6:00 PM (this prevents restarts during work hours).
Toggle Get me up to date to On.
Scroll down to Delivery Optimization and click it.
Review the setting for Allow downloads from other PCs — this controls whether the PC downloads updates from other devices on the local network (saves bandwidth) or only from Microsoft.
Screenshot: Capture the Advanced options page showing active hours and the Delivery Optimization settings.


Why this matters: Unmanaged updates can restart a workstation during a presentation or critical task. Setting active hours and delivery optimization are standard steps in any enterprise deployment.
